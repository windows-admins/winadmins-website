Tech Support

The WinAdmins Discord server is primarily geared towards Information Technology professionals managing enterprise environments. We have compiled a short list of resources available for end user tech support that you might find helpful in resolving problems.

  • The Microsoft Community Discord for all things Microsoft (https://aka.ms/community-discord)
  • Reddit’s /r/TechSupport Discord for all things technology (https://discord.com/invite/2EDwzWa)
  • Reddit’s /r/Apple Discord for all things Mac, iPhone, Apple Watch, etc. (https://discord.com/invite/apple)
